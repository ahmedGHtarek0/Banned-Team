## 🌟 Overview

**UniReserve** is a state-of-the-art management system tailored for educational institutions. It provides a seamless, high-performance experience for managing lecture halls, multi-purpose rooms, and institutional scheduling. With a focus on **Cinematic Dark Mode** and **Institutional RBAC**, it ensures that the right people have the right tools to manage university resources efficiently.

## 🚀 Key Features

- **🎬 Cinematic Dark Mode**: A premium, high-contrast dark aesthetic with smooth transitions and persistent user preferences.
- **🛡️ Institutional RBAC (Role-Based Access Control)**:
    - **Branch Manager (Faisel)**: Master authority with full CRUD capabilities across the platform.
    - **Admin (Akram)**: Specialized management for Lecture Halls; Multi-Purpose room proposals require higher authorization.
- **⏳ Ramadan Scheduling Engine**: Dynamic scheduling logic optimized for 55-minute intervals, supporting institutional requirements during peak periods.
- **📊 Global Data Export**: Export critical data (Bookings, User Lists, Hall Inventory) directly to CSV/Excel for administrative reporting.
- **✨ Micro-Animations**: A fluid user experience powered by **Framer Motion** and **Tailwind CSS**.
- **👥 Staff Delegation**: Real-time administrative tools to promote employees to Admin status instantly.

## 🛠️ Tech Stack

- **Frontend**: [React 18](https://reactjs.org/) & [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Data Handling**: [Date-fns](https://date-fns.org/)
- **State Management**: React Context API (Localized & Global Persistence)

## 📦 Installation & Setup

Get started with UniReserve in seconds:

```bash
# Clone the repository
git clone https://github.com/ahmedGHtarek0/Banned-Team.git

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```
