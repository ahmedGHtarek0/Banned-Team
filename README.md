# UniReserve - Advanced University Hall Management System

A cinematic, high-performance institutional room booking and management platform.

## 🚀 Key Features
- **Cinematic Dark Mode**: Seamless theme switching with persistent user preference.
- **Institutional RBAC**: 
    - **Branch Manager (Faisel)**: Full master authority for direct CRUD operations.
    - **Admin (Akram)**: Direct management of Lecture Halls; Multi-Purpose proposals require authorization.
- **Ramadan Scheduling Engine**: Instant 55-minute interval scheduling for the whole university.
- **Global Archive**: Functional CSV/Excel data export for bookings, users, and hall inventory.
- **Advanced Micro-Animations**: Premium UI interactions powered by Framer Motion and Tailwind CSS.
- **Real-time Staff Delegation**: Promote any employee to Admin status instantly.

## 🛠️ Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS
- **State**: AppContext for localized global data persistence
- **Icons**: Lucide React
- **Dates**: Date-fns

## 📦 Production
Built with Vite for high-performance delivery.
```bash
npm install
npm run dev # Local Development
npm run build # Production Bundle
```
